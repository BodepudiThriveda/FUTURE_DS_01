📌 Project Overview

I built an interactive Sales & Profit Dashboard using Tableau to provide actionable business insights. The dashboard helps in understanding:

Which products are top sellers

How sales and profits evolve over time

Regional and category performance

This makes it easier for businesses to spot growth opportunities, identify bottlenecks, and make data-driven decisions.

🔹 Key Features

Interactive Filters: Drill down by product, category, region, and time period

Sales & Profit Visualizations: Track trends, identify high-performing areas, and monitor performance gaps

Actionable Insights: Quickly identify areas for improvement and growth

💡 Skills Gained

This project strengthened my abilities in:

Data visualization and storytelling with Tableau

Creating interactive dashboards for business insights

Converting raw data into actionable, decision-ready information

🛠 Tools & Technologies

Tableau Desktop: Dashboard creation and visualization

Excel / CSV: Source data preparation
